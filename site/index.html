unofficial · not affiliated with Catppuccin or Gruvbox

Install

  1. Install the Stylus browser extension.
  2. Pick a theme below and download its file.
  3. In Stylus: Manage → Backup → Import the file. Every style installs at once.

Themes with a light variant auto-switch with your OS appearance.

Clear installed styles

Load the temporary Chrome helper extension from tools/stylus-cleaner, then click Preview and Delete matched styles.

The helper deletes exact marker matches and verified legacy matches only, downloads a backup first, and avoids DevTools paste commands.
