Gruvbox Userstyles

A Gruvbox recolor of the Catppuccin userstyles for Stylus.

unofficial · not affiliated with Catppuccin or Gruvbox

Install

  1. Install the Stylus browser extension.
  2. Pick a theme below and download its file.
  3. In Stylus: Manage → Backup → Import the file. Every style installs at once.

Themes with a light variant auto-switch with your OS appearance.

Clear installed styles

Download the repo ZIP, unzip it, load the temporary Chrome extension from tools/stylus-gruvbox-cleaner, then use Stylus Gruvbox Cleaner to preview and delete.

It is not a general Stylus cleaner: it matches only Gruvbox imports with the exact marker, or old imports with two repo signals, and downloads a backup first.
